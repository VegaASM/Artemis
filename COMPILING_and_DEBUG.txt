
How to compile Source (Linux)

Step 1. Install PowerPC Linux Compiler & QEMU
sudo apt-get update
sudo apt-get install gcc-aarch64-linux-gnu build-essential binutils-aarch64-linux-gnu binutils-aarch64-linux-gnu-dbg qemu-user qemu-user-static

Step 3: Build from Source Code
cd Source
aarch64-linux-gnu-gcc -ggdb3 -o main -static main.S

Step 4: Make file executable
chmod +x ./main

Step 5: Delete pre-compiled main
cd Artemis
rm main

Step 6: Final step
Move newly compiled main file into Artemis folder. Congratz.

--

Debugging:

qemu-aarch64 -g 1234 ./main

gdb-multiarch -q --nh \
  -ex 'set architecture aarch64' \
  -ex 'file main' \
  -ex 'target remote localhost:1234' \
  -ex 'break main' \
  -ex continue \
  -ex 'layout split' \
  -ex 'layout regs'
